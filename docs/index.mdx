# Introduction

The iopool integration for Home Assistant allows you to seamlessly connect and monitor your iopool EcO pool system within the Home Assistant platform.

This integration provides:
- Real-time monitoring of your pool parameters (temperature, pH, ORP, mode, action required)
- Automatic management of pool filtration (summer/winter modes)
- Boost mode for temporary increased filtration
- Full integration with Home Assistant entities and automations
- Multi-pool support (one config per pool)
- Advanced scheduling and statistics for filtration duration

With this integration, you can automate your pool maintenance, monitor water quality, and optimize filtration cycles directly from Home Assistant.

# Features

- **Sensors**: Temperature, pH, ORP, Mode, Filtration Recommendation, Action Required
- **Filtration Automation**: 
  - Summer and winter modes with configurable schedules and durations
  - Two summer slots with percentage-based duration split
  - Minimum and maximum duration constraints
- **Boost Mode**: Temporarily increase filtration for 1h, 4h, 8h, or 24h
- **Filtration State Tracking**: Real-time binary sensor reflecting pump state
- **History Stats**: Daily elapsed filtration duration sensor
- **Multi-language**: English and French translations
- **Home Assistant Integration**: All entities are available for automations, dashboards, and scripts
- **Diagnostics**: Downloadable diagnostics for support and debugging
